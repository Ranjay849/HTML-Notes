HTML (HyperText Markup Language)

  • HTML stands for HyperText Markup Language. It is the language used to make webpages.
    • HyperText: This refers to the ability to create links that connect web pages to one another, making the web a connected "web" of information.
    • Markup Language: This means you use "tags" to surround your content, giving that content meaning and structure. You are "marking up" a plain text document.
  • HTML is not a programming language; it is a markup language.
  • It works with tags (like <p> for paragraph, <h1> for heading) to tell the browser how to show the content.

The Heading (<h1> to <h6>)

  • HTML me heading ka use title ya section ke naam dene ke liye hota hai.
  • Headings ko <h1> se <h6> tags me likhte hain.
  • Headings SEO (Search Engine Optimization) me bhi important hote hain.
  • Page me ek hi <h1> hona chahiye (main title).
  • <h2> - <h6> subsections aur topics ke liye use karte hain.
  • <h1>: The most important heading, typically used only once per page for the main title.
  • <h2>: A major section heading.
  • <h3>: A sub-section heading under an <h2>.
  • ...and so on, down to <h6>, the least important heading.

The Paragraph (<p>)

  • Paragraph ka use text ko group karke likhne ke liye hota hai.
  • HTML me paragraph banane ke liye <p> tag use hota hai.
  • Browser automatically paragraph ke upar-neeche thoda gap (space) de deta hai.

Important Points:

  • Paragraph text is always written inside <p> ... <p>.
  • Browser ignores extra spaces and line breaks inside a paragraph (agar aap 10 space bhi daloge to browser usko ek hi space lega).

HTML Horizontal Rules (<hr>)

  • <hr> stands for Horizontal Rule.
  • It is used to insert a horizontal line across the page.
  • This line is often used to separate sections of content.
  • <hr> is a self-closing tag â†’ it does not need an ending tag.

The Line Break Tag (<br>)

  • <br> stands for Line Break. It is used to move text to the next line without starting a new paragraph.
  • It is a self-closing tag â†’ no closing </br> is required.

List Elements (<ul>, <ol>, <li>)

  • A list is a way to display a collection of related items on a webpage.
  • Lists make the content organized, easy to read, and structured.
  • In HTML, lists are created using special tags.

    There are mainly two types of Lists in HTML -

    1. Ordered List (<ol>)

      • An ordered list shows items in a numbered sequence (1, 2, 3 â€¦).

The Anchor Tag (<a>)

  • The anchor tag is written as <a>. It is used to create hyperlinks (clickable links).
  • With it, you can link:
    • One webpage to another
    • A section within the same page
    • An email address or phone number
  • To make text (or an image) clickable, allowing users to navigate. It requires an attribute called href (hypertext reference) to specify the destination URL.

HTML Links - The target Attribute

  • By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.
  • The target attribute specifies where to open the linked document.
  • The target attribute can have one of the following values:
    • _self - Default. Opens the document in the same window/tab as it was clicked
    • _blank - Opens the document in a new window or tab

The Image Tag (<img>)

  • The <img> tag is used to display images on a webpage.
  • It is a self-closing tag (no ending tag like </img>).
  • The most important attribute is src (source of the image).
  • src (source): The location of the image file (URL or local file).
  • alt (alternative text): Text shown when the image cannot load. Also useful for accessibility (screen readers).

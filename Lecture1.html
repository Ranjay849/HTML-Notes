What is Internet?

  • The Internet is a huge network of computers all over the world that can communicate with each other.
  • Computers follow rules called protocols to send and receive information safely and correctly.
  • Hum Internet ka use roz karte hain emails bhejne, friends se chat karne, videos dekhne, websites browse karne, online groups join karne, shopping karne, aur ghar ya office ke devices connect karne ke liye.

IP Address

  • IP stands for Internet Protocol.
  • IP Address ek unique address hai jo har computer ko Internet par allot hota hai.
  • When you connect your device to the Internet, the Internet service provider (ISP) assigns an IP address to your device. When you disconnect, that IP address can be given to someone else.

    There are two types of IP (based on assignment) -

    1. Static IP Address -

      • Assigned manually or reserved by your ISP.
      • It never changes, no matter how many times you connect.
      • It is like your permanent house that never changes.

    2. Dynamic IP Address -

      • Assigned automatically by your Internet Service Provider (ISP) every time you connect.
      • It changes when you disconnect and reconnect.
      • Dynamic IP is like a temporary house: someone stays for a while, then leaves, and another person can use it.

    There are two types of IP (based on network) -

    1. Public IP Address -

      • A Public IP address is your global, unique address on the internet.
      • Every device that accesses the Internet gets a unique public IP (assigned by your ISP).
      • It must be globally unique. No two devices on the internet can have the same public IP address at the same time.

    2. Private IP Address -

      • A Private IP address is a local address used only within your own private network (e.g., your home Wi-Fi, your office network).
      • Devices in the same network can see each other using private IPs, but these are not visible on the Internet.
      • Reserved Ranges: The internet authorities have reserved specific ranges of addresses just for this purpose. If you see an IP address starting with one of these, you know it's a private address:
        • 10.0.0.0 to 10.255.255.255 (Used by large corporations)
        • 172.16.0.0 to 172.31.255.255 (Used by medium-sized networks)
        • 192.168.0.0 to 192.168.255.255 (Most common for home networks)

    There are two versions of IP Address -

    1. IPv4 -

      • IPv4 stands for Internet Protocol version 4.
      • It uses 32-bit addresses, e.g., 192.168.1.1.
      • It can support about 4.3 billion unique addresses.
      • Most Internet devices still use IPv4.

    2. IPv6 -

      • IPv6 stands for Internet Protocol version 6.
      • It uses 128-bit addresses, e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334.
      • The number of possible IPv6 addresses is 2^128, which is roughly: 340,000,000,000,000,000,000,000,000,000,000,000,000 (340 undecillion)
      • It is created because IPv4 addresses were running out.

MAC Address

  • A MAC (Media Access Control) Address is a unique, permanent serial number burned into every network-capable piece of hardware (your laptop's Wi-Fi card, your phone, your smart TV, the network port on your desktop).
  • It's used to identify a device on a local network.
  • When devices communicate on the same network (like Wi-Fi or LAN), they use MAC addresses to make sure the data goes to the correct device.
  • A MAC address is a 48-bit number. To make it readable for humans, it's typically written as 12 hexadecimal digits.
  • The most common ways you'll see it displayed are:
    1. Colon-Separated (Most Common):
      2. 3C:22:FB:A3:B4:C5
    2. Hyphen-Separated (Common on Windows):
      3. 3C-22-FB-A3-B4-C5
    3. Period-Separated (Used by Cisco and other network gear):
      4. 3c22.fba3.b4c5
    4. No Separators (Less common, seen in some software):
      5. 3C22FBA3B4C5

DNS (Domain Name System)

  • DNS stands for Domain Name System. It is like a phone directory for the Internet.
  • It translates human-friendly domain names (like coderarmy.in) into IP addresses (like 172.21.30.11) that computers use to communicate.
  • When you type a website name in your browser, your computer asks a DNS server for the corresponding IP address.
  • The DNS server responds with the IP, so your computer can connect to the website.
  • Think of it like looking up a person's phone number in a phone book. You know the name, but to call them, you need the number.


Web Development

  • Web Development means creating and maintaining websites or web applications that run on the Internet.
  • It mainly has two sides:

    1. Frontend (Client-side):

      • This is what the user sees on their browser (design, layout, buttons, forms, images).
      • Built using HTML, CSS, JavaScript.
      • Example: The look and feel of a website like colors, navigation, or login page design.

    2. Backend (Server-side):

      • This is what happens behind the scenes (data storage, server logic, authentication).
      • Built using languages like PHP, Python, Java, Node.js, and databases like MySQL, MongoDB.
      • Example: When you log in, the backend checks your username/password from the database.

MERN Stack

  • The MERN Stack is a popular set of technologies used to build modern full-stack web applications.
  • It includes 4 main tools (M-E-R-N):
    • M: MongoDB
    • E: Express.js
    • R: React.js
    • N: Node.js

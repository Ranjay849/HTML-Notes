HTML Media

  • Websites are not just about text and forms.
  • Media (like images, audio, and video) makes web pages interactive and engaging.
  • HTML provides special tags to embed different types of media.

    1. Adding Images - <img>

      • The <img> tag is used to display an image on a webpage.
      • It is a self-closing tag and does not require a closing tag.

    2. Adding Audio - <audio>

    3. Adding Video - <video>

      • To play a video file in HTML, we use the <video> element:

      • width/height â†’ sets video size.

    4. Embedding External Content - <iframe>

      • The <iframe> tag is used to embed another webpage inside your webpage.
      • It is commonly used for YouTube videos, Google Maps, advertisements, or forms.

      • <iframe>: Embeds external content (like a YouTube video) on your webpage.
      • src="https://www.youtube.com/embed/NGYp1zCBthk?si=MtXdXTxheKd1wAfH": URL of the video to display.
      • width="500" height="300": Sets the size of the video frame.
      • frameborder="0": Removes the border around the iframe.
      • allowfullscreen: Lets the video be viewed in full-screen mode.
      • </iframe>: Closes the iframe element.

Multi-Page Website

  • A multipage website is a site made up of multiple linked HTML pages, each showing different information.
  • Each page has its own HTML file (like index.html, about.html, contact.html).
  • Pages are connected to each other using links (<a> tag).
  • Common example: Home, About Us, Services, Contact Us.
  • Used when website has a lot of information, which cannot fit into a single page.
  • Example:
    • Home
    • About
    • Skills
    • Projects
    • Services

Deploying a Website on Netlify

  • Netlify is a platform to host static websites easily and quickly.
  • It provides free hosting, automatic HTTPS, and continuous deployment from Git repositories.

    Steps to Deploy a Website on Netlify

    • Step 1: Prepare your website files
      • Ensure your project has an index.html file.
      • Include all CSS, JavaScript, and image files in proper folders.

    • Step 2: Create a Netlify account
      • Sign up at Netlify using email or GitHub/GitLab/Bitbucket.

    • Step 3: Deploy the website
      • Click "Add new site" â†’ "Deploy manually".
      • Drag and drop your website folder containing index.html.

    • Step 4: Configure your site
      • Netlify will generate a random URL; you can customize the site name later.
      • Check the site URL to confirm your website is live.

    • Step 5: Optional - Connect GitHub for continuous deployment
      • Link your GitHub repository to Netlify for automatic updates on every push.

    • Advantages of Using Netlify

    • Free hosting for small projects.
    • Automatic HTTPS (SSL).
    • Simple drag-and-drop deployment.
    • Continuous deployment with Git integration.
